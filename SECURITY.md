# Security Policy

## Supported Versions

Security updates are provided for the latest `main` branch.

## Reporting a Vulnerability

Please use GitHub's private vulnerability reporting flow:

1. Go to the repository **Security** tab.
2. Select **Report a vulnerability**.
3. Include reproduction details, impact, and any suggested mitigation.

If private reporting is unavailable, open a minimal public issue without sensitive details and request a private contact channel.
